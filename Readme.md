#Skincare Web Application

##Project Overview

This is a skincare web application designed to help users understand their skin type, explore skincare solutions, and book personalized skincare consultations.

The project focuses on providing a simple, user-friendly interface for users to interact with skincare information and services.

##Features

* Skincare tips and product recommendations
* Booking form for appointments
* Responsive design (works on mobile & desktop)
* Navigation between pages (Home, Skincare, Booking,Consultation,Cart)

##Technologies Used

* HTML5
* CSS3
* JavaScript
* APIs for booking form

##Installation & Setup

1. Clone the repository:
...bash
git clone https://github.com/lysaomondi/spacklecare.git

2. Navigate into the project folder:

...bash
cd skincare

3. Open `index.html` in your browser.

##Testing

Unit testing is implemented using Jest to ensure core functions (such as skin type detection and form validation) work correctly.

Run tests using:
...bash
npm test
